##Mechanic
-My mechanic is Sword Combo, when left clicking the player will attack and if left click is pressed again during an attack animation, the player will then perform another animation (up to 3). 

##CONTROLS
-Movement - WASD
-Camera Control - Mouse
-Sprint - LShift
-Jump - Spacebar
-Attack/Combo - Left Click


##Key Areas
-Player State machine 
- Animation
- Attacks and combos
- stamina
- Target Dummy Attack feedback, shake sound and particle
